C#

The Language C# was Developed while Net was under development Before the development of C# Net was using a language called "Simply Managed C". In 1999 Anders Heilsberg Started development on C# but at the time it was called "C.O.O.L" C-Like Object Orented Language The name was considered to be kept but Didn't due to trademark reasons. Once NET was announced in 2000 C# was given it's current name "C#" the class and net libraries had also been ported to C#. The Designer of C# is Anders Heilsberg and had developed prior Turbo Pascal, Emberacado Delphini, and also J++. Many people Had stated that C# is a Java Clone but Anders said that C# is more closely related to C++ instead. In 2005 C# 2.0 was released
