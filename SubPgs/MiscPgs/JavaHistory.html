Java

James Gosling, Mike Sheridan, and Patrick Naughton the original creators of Java started the project in 1991. Java was originally designed for Interactive television but was too advanced for the cable at the time originally the project was called Oak but was changes to Green then finally Java. Java was designed after C and C++ to introduce familiarity to C and C++ programmers. Sun Microsystems released Java 1.0 in 1996 it was promised that you only need to Write Once, Run Anywhere (WORA is their slogan) it would provide no cost run-times and fairly good security that could be customized.

Webbrowsers would soon icorperate java applets within webpages. the java compiler was rewriten to align with the Java Specifications. Soon in 1998 Java 2 was released to the public which had more configurations for multiple platforms and it split into 3 types J2SE (Java 2 Standard Edition), J2ME (Java 2 Micro Edition), and J2EE (Java 2 Enterprise Edition). They also have a mascot for Java Named Duke.
